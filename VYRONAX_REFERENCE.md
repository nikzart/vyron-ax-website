## Ongoing Projects & Products

### IgnizeAI - UPSC Exam Preparation Platform
**Overview:** Personalized AI-powered learning platform for UPSC (Union Public Service Commission) exam preparation, specifically designed for Indian students.

**Key Features:**
- **Personalized Learning:** Dynamic context adaptation for each individual student
- **Custom LLM Integration:** Uses OpenAI API schema with specialized training
- **Accuracy Verification:** Built-in fact-checking and validation systems
- **RAG-Based Knowledge Base:** Complete textbook and note content stored in vector database
- **UPSC-Specific Answer Structure:** Follows official UPSC answer formats and requirements
- **Keyword Optimization:** Ensures proper terminology usage in answers
- **Admin Panel:** Complete platform management including book ingestion and vector DB management
- **Auto-Evaluation System:** Upload answer sheets, select exam date, automatic evaluation with reference to question papers
- **Web Search Integration:** Automatic question paper retrieval when not available locally
- **Self-Evaluation Framework:** Agentic approach to assessment
- **AI Companion:** Dynamically adjusts tone, complexity, and continuity based on student needs
- **Function Calling:** Progress tracking, analytics, and tools integration
- **Mathematical Evaluation:** Specialized math problem solving and evaluation
- **UML to Diagram Tools:** Visual learning aids
- **Web Search & Browsing:** Real-time information retrieval
- **MCP Integration:** Model Context Protocol implementation
- **Client-Side Vector Storage:** Browser-based local vector database for uploaded documents
- **Document Management:** Add/remove files from memory
- **Multi-Format Support:** PDFs, docs, text files, and images
- **Chat History:** Complete conversation tracking and context retention

**Technology Stack:**
- Custom LLM (OpenAI API compatible)
- Vector Database (RAG implementation)
- Function Calling & Agentic Framework
- Model Context Protocol (MCP)
- Client-side storage optimization

**Future Expansion:**
- CAT (Common Admission Test) preparation
- IIT-JEE exam coaching
- Full-Maths specialized model
